This project focuses on predicting breast cancer survival using a healthcare dataset (SEER Breast Cancer dataset). The main objective is to analyze various clinical and demographic factors affecting patient survival and build a machine learning model to make accurate predictions.
--> Dataset :
The dataset includes information such as age, tumor size, cancer stage, hormone receptor status, and survival months. (4000+ entries, multiple medical attributes)
--> Data Preprocessing :
Handled missing values
Removed unnecessary columns
Encoded categorical variables using Label Encoding
Performed feature selection
Prepared data for model training
--> Models Used :
Decision Tree Classifier
Logistic Regression
--> Model Overview :
Split dataset into training and testing sets
Trained multiple models for comparison
Evaluated performance using accuracy score
Used confusion matrix for analysis
Selected best model based on performance
--> Key Learnings :
Understanding healthcare datasets
Importance of data preprocessing
Handling categorical variables
Model comparison and evaluation
Interpreting machine learning results
